## EnhancedImageView


一个增强的自定义ImageView,具备手势放大缩小等功能,主要原理的是 `Matrix` + `ScaleGestureDetector` + `GestureDetector` 进行对图片进行移动与裁剪

-------------------

- 单指滑动
- 多指滑动
- 双击放大(2倍)
- 放大状态双击恢复
- 手势放大
- 无与ViewPager滑动冲突
